# Constraints

V1 constraints:

- No authentication
- No payments
- No hardware
- No sensors
- No image upload
- No Kubernetes
- No PostgreSQL requirement
- No mobile app
- No community features

Product constraints:

- Must be usable by beginners.
- Must not overload user with values.
- Must prioritize actions.
- Must show uncertainty.
- Must tell the user what not to do.

Technical constraints:

- Local-first development.
- Docker Compose must run the full app.
- SQLite is the default database.
- AI must be optional and disabled if no API key exists.
